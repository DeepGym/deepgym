# Anagram Check

Write a function that checks if two strings are anagrams of each other. Consider only lowercase letters.

## Function Signature

```python
def is_anagram(s: str, t: str) -> bool:
```

## Parameters
- `s` — first string
- `t` — second string

## Returns
- `True` if `s` and `t` are anagrams, `False` otherwise

## Examples

```
is_anagram("anagram", "nagaram") -> True
is_anagram("rat", "car") -> False
is_anagram("", "") -> True
```
