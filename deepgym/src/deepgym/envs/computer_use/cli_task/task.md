# CLI Task — Git Operations

Use git to initialize a repository and perform basic operations.

## Instructions

Write a Python script that:

1. Create a directory `/tmp/my_repo` (remove it first if it exists).
2. Initialize a git repository in `/tmp/my_repo`.
3. Create a branch named `feature`.
4. Create a file `hello.txt` with the content "Hello, DeepGym!".
5. Stage and commit the file with the message "Add hello.txt".
6. Print "done" to stdout when complete.
