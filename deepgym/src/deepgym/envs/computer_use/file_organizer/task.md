# File Organizer

Create a directory structure and populate it with files.

## Instructions

Write a Python script that:

1. Create the following directory structure under `/tmp/project`:
   - `src/`
   - `tests/`
   - `docs/`

2. Create a `README.md` in each directory with the following content:
   - `src/README.md`: "# Source Code\nMain application source files."
   - `tests/README.md`: "# Tests\nUnit and integration tests."
   - `docs/README.md`: "# Documentation\nProject documentation and guides."

3. Print "done" to stdout when complete.
