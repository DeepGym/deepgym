# Reverse String

Write a function that reverses a string.

## Function Signature

```python
def reverse_string(s: str) -> str:
```

## Parameters
- `s` — a string to reverse

## Returns
- The reversed string

## Examples

```
reverse_string("hello") -> "olleh"
reverse_string("Python") -> "nohtyP"
reverse_string("") -> ""
reverse_string("a") -> "a"
```
