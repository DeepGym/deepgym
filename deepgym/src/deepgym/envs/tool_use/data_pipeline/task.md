# Data Pipeline

Read a CSV file, filter rows, and write the result.

## Instructions

Write a Python script that:

1. Read `input.csv` from the current working directory.
2. Filter rows where the `age` column value is greater than 30.
3. Write the filtered rows (with headers) to `output.csv` in the current working directory.
4. Print the number of filtered rows to stdout.

The CSV has columns: `name`, `age`, `city`.
